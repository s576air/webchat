# webchat
간단한 웹 채팅 서버입니다.

구현된 기능
1. 로그인 및 회원가입이 가능합니다.
2. 친구코드를 통해 친구 추가가 가능합니다.
3. 원하는 친구를 추가해 채팅방을 만들 수 있습니다.
4. 텍스트, 이미지, 영상, 소리 등의 미디어 파일을 전송 가능합니다.

참고사항
1. db는 h2 database를 사용하며 메모리 모드이기 때문에, 서버를 종료하면 기록된 내용도 사라집니다.
2. 서버를 중점적으로 만들었기 때문에, 프론트엔드가 미흡합니다.
3. 테스트용 아이디는 다음과 같습니다.

- id: test, test2, test3
- password: (공통) password
참고 링크: https://github.com/s576air/webchat/blob/main/src/main/java/com/s576air/webchat/DataInitializer.java


하드코딩에 관련하여 오해가 있을 수 있는 것 같아 추가 정보를 작성합니다.
우선 서버는 http://localhost:8080 을 통해 접속할 수 있습니다.
여러 유저에게 적용되는 변경사항(친구추가, 채팅방 개설, 채팅 등)은 다른 브라우저에 다른 아이디로 로그인하여 확인할 수 있습니다.
1. 해당 프로그램은 회원가입 기능이 구현되어 있으며,
http://localhost:8080/login 의 왼쪽 아래에 파란 '회원가입' 하이퍼링크를 눌러 이동할 수 있습니다.
현재 새로 회원가입한 회원의 닉네임은 default로 설정됩니다.
2. 친구 코드는 알고리즘에 의해 회원가입시마다 생성됩니다.
sqids 라이브러리를 사용하며, 앞부분 8자리는 암호화된 회원가입 순서(암호학적으로 안전하지 않음), 뒷부분 8자리는 랜덤하게 생성됩니다.
3. 채팅방은 http://localhost:8080/chatroom-list 의 맨 위의 파란 '새 채팅방 만들기' 하이퍼링크를 통해 추가할 수 있으며, 다른 유저의 경우 새로고침 후 채팅방이 추가되는 것을 확인할 수 있습니다.
4. 채팅 기능은 웹소켓을 사용하며 새로고침 또는 버튼 입력 없이 다른 유저에게 실시간으로 메시지를 전송할 수 있습니다.
또한 파일을 끌어다놓거나 선택하여 미디어 데이터를 전송할 수 있으며, 비디오, 오디오, 이미지를 실시간으로 전송할 수 있습니다.
